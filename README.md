# Blockchain Fraud Detection Analysis

## Overview
This project analyzes survey-based numerical data (n = 113) to evaluate how blockchain-based data analytics improves fraud detection and financial transaction security.

## Dataset
- blockchain_survey_data.xlsx  
  (Likert-scale responses from finance, banking, and IT professionals)

## Methods
- Descriptive statistics
- Correlation analysis
- Regression analysis

## Key Insight
Real-time monitoring and fraud pattern detection show a strong positive impact on financial security.
