# Analysis Summary

## Objective
To evaluate whether blockchain-based data analytics improves financial transaction security and fraud detection.

## Dataset
- blockchain_survey_data.xlsx
- Sample size: 113 respondents
- Data type: Likert-scale (1 = Strongly Disagree, 5 = Strongly Agree)

## Descriptive Analysis
Mean scores for key variables were above 4.0, indicating strong agreement that:
- Blockchain improves transaction security
- Blockchain increases financial transparency
- Blockchain analytics supports fraud detection

## Correlation Analysis
A strong positive correlation was observed between:
- Real-time monitoring and fraud prevention
- Transaction transparency and fraud detection

This suggests that increased visibility of transactions enhances the ability to identify fraudulent activity.

## Regression Analysis
Regression results show that:
- Fraud pattern detection
- Real-time transaction monitoring

are statistically significant predictors of improved financial security.

## Conclusion
The analysis confirms that blockchain-based data analytics plays a significant role in strengthening financial security and preventing fraud.
